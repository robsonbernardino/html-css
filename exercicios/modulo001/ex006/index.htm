Principais Formatações

  • Negrito/Destaque

    Para colocar a palavra em negrito usamos a tag "strong"

  • Itálico/Ênfase

    Para frases ou palavras em itálico, utilizamos a tag "em"

  • Texto marcado

    Podemos criar também um texto marcado utilizando a tag "mark"

    E se eu tiver várias marcações, e quiser uma cor diferente em uma delas, basta colocar "style" dentro da abertura da tag com a cor desejada

  • Texto grande e pequeno

    Vamos criar um texto grande e um texto pequeno. utilizando as tags "big" ou "small".

  • Texto Deletado

    Podemos marcar um texto como deletado utilizando a tag "del". Isso indica que o texto deve ser lido porém não deve ser considerado.

  • Texto inserido

    Podemos marcar um texto como inserido utilizando a tag "ins". serve para dar ênfase e indicar que foi adicionado depois.

    • Texto sobrescrito

    Para inserir coisas do tipo x20+3 utilizo a tag "sup" de superior.

    Texto subscrito

    Para inserir coisas do tipo H2O utilizo a tag "sub" de subscrito.
