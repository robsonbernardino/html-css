Lista de definições

HTML
É a estrutura de uma página web. Com o HTML, você define os elementos da página, como títulos, parágrafos, imagens, botões, links, etc. Pense nele como o esqueleto de um site — é o que dá forma e organiza o conteúdo.
CSS
É a aparência do site. Com o CSS, você estiliza os elementos criados no HTML, escolhendo cores, tamanhos, fontes, espaçamentos, posicionamentos e até efeitos visuais. Ele transforma aquele esqueleto básico em algo bonito e profissional.
JavaScript
É a inteligência e o comportamento da página. Com o JavaScript, você adiciona interatividade, como responder a cliques, validar formulários, mostrar mensagens, criar animações, carregar conteúdo sem recarregar a página, entre outros. Ele dá vida ao site.
